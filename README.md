# Slither Pro Live

Compact Render-ready multiplayer HTML game.

Files:
- index.html
- server.js
- package.json
- render.yaml
- README.md

Render settings:
- Environment: Node
- Build Command: `npm run build`
- Start Command: `npm start`
- Root Directory: leave blank

This version uses a custom zero-dependency WebSocket server, so Render does not need to install Express, Socket.IO, ws, or node_modules.
