IMPORTANT: Your Render dashboard is still set to the wrong Build Command: rpm install

Use exactly these Render settings:

Runtime: Node
Build Command: npm ci
Start Command: npm start
Root Directory: leave blank if this folder is the repo root
Node Version: 20.20.2

If Render still says "Running build command 'rpm install'", it is using old dashboard settings, not the files. Open Render > Settings > Build & Deploy and replace rpm install with npm ci.
